# DPO 训练过程曲线分析报告

## 1. 实验背景概述

本实验采用 **Direct Preference Optimization (DPO)** 方法，对语言模型进行偏好对齐训练。
训练目标是：

> 在不引入显式 reward model 的情况下，使模型对 *chosen responses* 给出更高概率、对 *rejected responses* 给出更低概率。

以下分析基于完整训练过程中记录的 rewards、log-probabilities、logits、loss、梯度与学习率等指标。

------

## 2. 偏好学习核心指标分析（DPO 是否成功）

### 2.1 `train/rewards/chosen` 与 `train/rewards/rejected`

**含义**

- `rewards/chosen`：模型对人类偏好回答的打分
- `rewards/rejected`：模型对被拒绝回答的打分

**观察结果**

- `rewards/chosen` 稳定在 **约 6.8 – 7.6 区间**
- `rewards/rejected` **持续单调下降**，从 ~2.8 降至接近 0 / 负值

**结论**

- 模型在训练过程中**并未通过简单抬高所有回答的 reward**
- 而是明确地 **压低 rejected response**
- 这是 DPO 区别于 SFT 的关键特征，表现健康

------

### 2.2 `train/rewards/margins`（最关键指标）

**定义**

```
margin = reward(chosen) − reward(rejected)
```

**观察结果**

- margin 从 ~4.3 持续增长至 ~7.5
- 整体呈明显上升趋势，后期维持高位，轻微震荡

**结论**
✅ **偏好分离持续增强**
该指标直接反映 DPO 的优化目标，本实验中：

- 模型对偏好排序的信心不断增强
- 无 collapse 或退化迹象
  👉 可视为 **DPO 训练成功的最强证据**

------

### 2.3 `train/rewards/accuracies`

**含义**

- 在训练对中，模型是否正确给 chosen 更高 reward

**观察结果**

- 准确率长期维持在 **0.97 – 1.00**
- 偶发下降至 ~0.95，但迅速恢复

**结论**

- 偏好判断已接近饱和
- 后期波动主要来自 batch 随机性，而非性能退化

------

## 3. 概率层面分析（log-probability）

### 3.1 `train/logps/chosen` vs `train/logps/rejected`

**含义**

- log-prob 是模型对整段 response 的条件对数概率
- 可跨 step、batch、模型比较

**观察结果**

- `logps/rejected`：整体 **持续下降（更负）**
- `logps/chosen`：保持相对稳定，仅存在正常波动

**结论**

- 模型正在**系统性地降低 rejected responses 的生成概率**
- 而非牺牲 chosen responses 的 likelihood
- 符合 DPO “push apart” 的设计目标

------

## 4. logits 层面分析（对比用，非训练目标）

### 4.1 `train/logits/chosen` 与 `train/logits/rejected`

**观察结果**

- 两者均呈现 **缓慢下降趋势**
- chosen 与 rejected 的 logits 差距并不显著扩大

**解读**

- logits 是 softmax 前的原始打分，**不具备全局可比性**
- DPO 的优化并不直接作用于 logits，而是作用于 log-prob 的差值
- 因此 logits 不“拉开”是 **完全正常现象**

⚠️ 这也再次说明：

> **评估 DPO 成功与否，应看 log-prob / reward / margin，而不是 logits**

------

## 5. 优化稳定性分析

### 5.1 `train/loss`

**观察结果**

- loss 保持在 **0.02 – 0.06 的低区间**
- 存在尖峰，但无持续上升或发散

**结论**

- DPO loss 本身非单调目标
- 当前 loss 形态表明：
  - 训练稳定
  - 未出现数值不稳定或偏好翻转

------

### 5.2 `train/grad_norm`

**观察结果**

- 大多数 step 中梯度范数 < 2
- 偶发峰值（~3–5），但迅速回落

**结论**

- 梯度整体受控
- 峰值通常来自：
  - 较难区分的 preference pair
  - margin 接近决策边界的样本
- 未观察到梯度爆炸风险

------

### 5.3 `train/learning_rate`

**观察结果**

- 学习率随 step **线性衰减**
- 与 loss、grad_norm 的行为一致

**结论**

- 学习率调度合理
- 后期训练更加平滑，有助于稳定收敛

------

## 6. 训练阶段性总结

| 维度              | 结论             |
| ----------------- | ---------------- |
| 偏好分离          | ✅ 明显、持续增强 |
| DPO 核心目标      | ✅ 达成           |
| 概率层行为        | ✅ 合理           |
| 数值稳定性        | ✅ 良好           |
| 过拟合迹象        | ❌ 未观察到       |
| 是否可 early stop | ✅ 后期已趋于稳定 |

------

## 7. 总体结论（一句话版）

> 本次 DPO 训练过程稳定、有效，模型成功学习到人类偏好排序，显著拉开 chosen 与 rejected responses 的概率差距，且未引入训练不稳定或模式退化问题。
